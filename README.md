## Dependencies
* Python
** python-jira
** python-configparser
* config.txt file in this directory containing the currencies you want to trade containing:
```
[BASICAUTH]
JIRA_USER = <user>
JIRA_PASS = <pass>
JIRA_URL = <url>
```

### How to install on Ubuntu
```
sudo apt update
sudo apt install python python-jira python-configparser
```
